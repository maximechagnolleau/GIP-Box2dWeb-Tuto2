# Game in Progress

## Box2d Web - Tuto 2 : move your body !

### Présentation

Ce projet permet d'appréhender le déplacement des objets 2D et de mettre en place la gestion des contrôles utilisateur dans un environnement Box2d Web.
Il présente les fonctionnalités suivantes :
* Ajouter la gestion des contrôles utilisateur,
* Simuler les déplacements et sauts d’un personnage à l’écran.
